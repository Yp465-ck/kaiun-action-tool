# 開運アクション生成エンジン

## 最終仕様 v2.1

---

## 0. 基本思想

暦は「加点方式」ではなく、

**優先 → 抑制 → 補正 → 統合**

で判断する。

---

## 1. 判定フロー（アルゴリズム本体）

### STEP1：強い天体イベント確認

該当する場合は主軸候補に昇格：

- 新月（当日）
- 満月（当日）
- 上弦・下弦（当日）
- 逆行開始・終了日

※通常日は補助扱い

---

### STEP2：十二直の本来の意味を確認

その日の「地上エネルギーの軸」を決める。

| 十二直 | 本来の意味（動詞で翻訳） |
|--------|--------------------------|
| 建 | 始める・立ち上げる |
| 除 | 取り除く・外す |
| 満 | 満ちる・満たす |
| 平 | 平らにする・整える |
| 定 | 定着させる・決める |
| 執 | 固める・継続する |
| 破 | 突破する・切る |
| 危 | 慎重に扱う・確認する |
| 成 | 実らせる・成就させる |
| 納 | 受け取る・収める |
| 開 | 開く・通じさせる |
| 閉 | 閉じる・守る |

※単語化しない。必ず「動詞」に翻訳する。

---

### STEP3：強い吉凶日の性質を確認

吉凶日は「方向補正」に使う。

**吉日の性質：**
| 吉凶日 | 性質（方向補正） |
|--------|------------------|
| 一粒万倍日 | 育つ・広がる |
| 天赦日 | 許される・始められる |
| 甲子の日 | サイクルの原点・ゼロから始める |
| 寅の日 | 循環・出て戻る |
| 巳の日 | 弁財天の縁日・金運整える |
| 大安 | 万事吉 |

**凶日の性質（抑制要素）：**
| 吉凶日 | 性質（抑制） |
|--------|--------------|
| 十死日 | 勢いNG・小さく |
| 不成就日 | 新規抑制・確認優先 |
| 仏滅 | 控えめに |

---

### STEP4：凶要素があれば"抑制"をかける

凶要素がある場合、主軸の方向を抑制する。

**例：**
- 火強 + 建 + 十死日 → 「始める」だが「小さく」
- 満 + 不成就日 → 「増やさず今あるものに目を向ける」
- 開 + 十死日 → 「開くが慎重に」

---

### STEP5：五行で"強弱とリスク"を確認

五行は「主軸変更」ではなく、強める・弱める・リスクを添えるに使う。

| 五行 | 強める方向 | リスク |
|------|------------|--------|
| 木強 | 伸びる・成長 | 伸びすぎ注意 |
| 火強 | 勢い・行動 | 勢いでやりすぎ注意 |
| 土強 | 安定・土台 | 動けなくなる注意 |
| 金強 | 切る・決断 | 切りすぎ注意 |
| 水強 | 内省・流れ | 停滞注意 |

---

### STEP6：統合（ここが最重要）

出力は **「動詞の一文」** にまとめる。

**抽象版（ロジック確認用）：**
- 抽象動詞OK（進める、向き合う、始める）
- 主軸の翻訳を確認するため

**具体版（Instagram用）：**
- 5分以内で完了する観測可能な行為
- 「片付ける」「捨てる」「書く」など

---

## 2. 主軸決定ルール（明文化）

### 通常日
十二直が主軸

### 強い月相当日（新月・満月・上弦・下弦）
月相を主軸候補に昇格し、十二直と統合

### 凶日がある場合
方向を抑制する

---

## 3. 矛盾調整の原則

| 状況 | 解決法 |
|------|--------|
| 増やす + 凶 | 小さく増やす |
| 満 + 抑制 | 今あるもので満たす |
| 建 + 火強 | 小さく始める |
| 危 + 一粒万倍 | 確認してから始める |
| 閉 + 一粒万倍 | 昨日始めたことを続ける |
| 開 + 十死日 | 慎重に開く |

**矛盾は消さない。中間解を出す。**

---

## 4. 月相扱いの最終定義

| 状況 | 扱い |
|------|------|
| 通常日 | 補助 |
| 新月・満月当日 | 主軸候補に昇格 |
| 上弦・下弦当日 | 主軸候補に昇格 |
| 前日・翌日 | 方向補助のみ |
| 逆行開始・終了日 | 主軸候補に昇格 |

---

## 5. 出力項目

### 1. その日の暦情報
日付を選択すると自動で表示。

```
📅 2026年2月21日（土）

【月相】二十六夜月
【十二直】建（たつ）
【日干支】丙寅（五行: 火）
【九星】四緑木星
【吉日】寅の日
```

### 2. 抽象版アクション（ロジック確認用）
主軸の翻訳を確認するための1行。

```
未来につながることを、小さく始める。
```

### 3. Instagram用ストーリー文章（コピペでそのまま使える）
具体版アクション入りの完成形。

```
🐯 今日の開運アクション
気になっていたことに、5分だけ手をつける。

今日は寅の日。
動いた分だけ巡りが生まれます。
勢いが強いので大きく賭けず、
小さく始めるのが◎💰
```

### 4. 裏ロジック解説
6ステップの判定フロー。

```
【判定フロー】
STEP1: 天体イベントなし → 十二直が主軸
STEP2: 建 → 「始める・立ち上げる」
STEP3: 寅の日 → 「循環」（方向補正）
STEP4: 凶要素なし
STEP5: 火強 → 「勢いリスク」
STEP6: 統合 → 「小さく始める」
```

## 6. 文章生成ルール

### 構造
```
[月相アイコン] 今日の開運アクション
[具体的アクション 15〜20文字（観測可能な行為）]

[エネルギー翻訳 1行]
[因果接続 1〜2行]
[ハードル下げ]
[締め + 絵文字]
```

### 動詞のルール
- 抽象版: 抽象動詞OK（進める、向き合う、始める）
- 具体版: 観測可能な行為のみ（片付ける、捨てる、書く）
- 5分以内で完了できること

---

## 7. 品質フィルター（最終チェック）

出力前に必ず確認：

- [ ] 5分以内でできるか？
- [ ] 誰にでも当てはまるか？
- [ ] 主軸の本来意味を守っているか？
- [ ] 凶要素を無視していないか？
- [ ] 断定予言していないか？

---

## 8. 文章に使ってOK / 使わない

### 使ってOK（有名・ポジティブ）
- 新月・満月・上弦・下弦
- 一粒万倍日・天赦日・甲子の日
- 寅の日・巳の日
- 大安
- 惑星逆行（水星逆行など）

### 使わない（ロジックのみ）
- 十二直の名前（建の日、危の日など）
- 二十八宿・九星
- 仏滅・不成就日・十死日（ネガティブ印象）

---

## 9. 禁止事項（最終版）

- 「縁起が良いからやる」
- 「金運アップだから投資」
- 単語主軸（増やす日、整える日）
- 十二直の名前を本文に出す
- 断定未来予測（「〜になります」）
- 投資・大きな出費を無条件で推奨
- 「静かに」など不自然な表現
- 抽象動詞のみでInstagram文章を終わらせる

## 10. 出力例

### 2/20（金）閉 + 一粒万倍日 + 木強

**抽象版アクション：**
```
昨日までに始めたことを、もう一歩進める。
```

**Instagram用：**
```
🌱 今日の開運アクション
やりかけのタスクを、1つ片付ける。

今日は一粒万倍日。
小さな種が大きく育つ日。
新しく始めるより、
続けることで芽が出やすくなります🌱
```

**裏ロジック：**
```
STEP1: 天体イベントなし → 十二直が主軸
STEP2: 閉 → 「閉じる・守る」
STEP3: 一粒万倍日 → 「育つ」（矛盾）
STEP4: 凶要素なし
STEP5: 木強 → 「伸びる」
STEP6: 矛盾調整 → 「新規ではなく既存を育てる」
```
